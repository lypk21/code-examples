1. the practised project is using Vue.js 2.6 to implement a small e-commence function for restaurant, with food menu listing, shopping cart and comments functions.

2. the tech stack are mainly Vue.js, axios, cube ui and stylus css pre-processor.

3. the UI is for mobile, so maybe open in mobile or use inspect it in chrome mobile view will get a better user experience.

4. how to run:
  1) npm install
  2) npm run serve
